# JS文件批量格式化工具使用说明文档

## 一、工具简介
本工具用于批量格式化指定文件夹中所有JavaScript（.js）文件，支持自定义缩进规则、换行保留等格式化参数，且会自动跳过`node_modules`目录（避免处理依赖文件）。格式化功能基于成熟的`jsbeautifier`库，确保代码格式化的规范性。

## 二、环境准备

### 1. 依赖安装
工具依赖`jsbeautifier`模块，使用前需先安装：
```bash
pip install jsbeautifier
```

### 2. 运行环境
- Python 3.6 及以上版本
- 支持 Windows、macOS、Linux 系统
